T2S 隱私權政策

本隱私政策描述了 T2S 應用程式如何以及何時收集、處理、使用和共享您的資訊。

使用我們的應用和服務，即表示您同意我們按照本政策所述收集、使用和共享您的資訊。

1 收集的資訊以及其用途

1.1 出於提升服務質量目的收集的資訊

出於改進服務質量，修復錯誤和了解應用使用情況的目的，本應用會收集以下資訊：

  • 錯誤資訊：一般錯誤資訊或程式錯誤堆疊，用於瞭解和修復應用程式中存在的錯誤。
  • 裝置資訊：如裝置型號，作業系統版本，系統語言，系統記憶體使用情況。
  • 效能資訊：應用啟動時間、幀生成時間、耗電量等。
  • 基本使用情況：使用者經常使用哪些功能，以及是否成功完成某項操作。
  • 朗讀引擎資訊：瞭解使用者使用的朗讀引擎服務資訊。
  • 可重置裝置識別符號：瞭解錯誤或操作是否由同一裝置產生。
  • 大致地理區域：透過 IP 地址瞭解使用者來自哪個國家或地區。只作彙總統計。

這些資訊不與您的個人身份資訊關聯，且不包含敏感資訊。 只能且只會用於提升服務質量。

本應用使用 Google Analytics for Firebase 和 Firebase Crashlytics 收集這些資訊。

1.2 基於興趣的廣告（適用於免費版使用者）

Google Mobile Ads SDK 可能會收集和共享以下資料型別，目的是用於基於興趣的廣告、分析和欺詐防範：

  • IP 地址：收集裝置的 IP 地址（可用於估測裝置所在的大致位置）。
  • 使用者如何與廣告互動。
  • 診斷資訊：收集與應用和 SDK 的效能相關的資訊，包括 崩潰日誌、應用啟動時間、掛起率和耗電量。
  • 裝置和帳號識別符號：收集 Android 廣告 ID、應用組 ID，如適用的話還會收集裝置上的登入賬戶有關的其他識別符號。

1.3 交易/賬單資訊

根據您購買的訂閱或付費服務，我們的支付網關合作夥伴可能會收集資訊，包括但不限於您的信用卡/借記卡詳細資訊或其他銀行帳戶詳細資訊以及您的賬單地址。 此資訊由支付網關合作夥伴儲存。例如，服務將使用第三方支付網關合作夥伴來處理和促進您向服務支付訂閱費，以便為您提供訂閱訪問許可權；此類支付網關合作夥伴不受我們控制。 當您訪問此類支付閘道器時，您將自行承擔風險。我們鼓勵您在向此類支付閘道器提供有關您的資訊或與它們進行任何交易之前，先熟悉此類支付閘道器提供的隱私宣告。 對於此類支付閘道器、此類支付閘道器的內容及其隱私慣例，我們不承擔任何責任或義務。

我們可能會從第三方供應商處收到財務或交易資訊（例如交易 ID 等）以用於付款、對賬和驗證交易以及解決投訴（如果交易失敗和退款）。

1.4 由使用者的操作提供給第三方的資料

在您使用本應用的過程中，您的操作行為可能會提供個人資訊給第三方。這不是我們能控制的，您應該對此自行負責。

我們建議你注意以下幾點。

1.4.1 網路瀏覽器

如果您使用本應用的內建瀏覽器瀏覽第三方網站，那麼第三方網站可能會收集或分享您的個人資訊。

1.4.2 朗讀引擎

本應用使用安裝在您裝置上的第三方的朗讀引擎（文字轉換語音服務），在您使用過程中本應用會把文字內容（可能是您輸入的內容，開啟的檔案，或瀏覽的網頁）提交給第三方服務來轉換音訊。
我們建議您檢視您使用的朗讀引擎提供商的隱私協議，並只使用您信任的提供商的朗讀引擎服務。

1.4.3 儲存或備份資料到第三方服務

如果您選擇透過第三方服務儲存或備份應用資料，那麼第三方服務提供者可能會收集或分享您的資訊。

1.5 其它情況

當您提供反饋、回覆調查或與我們溝通時，我們可能會收到你的個人資訊，比如您的姓名、電子郵件 ID 等。

我們使用第三方服務來括託管我們的各種部落格和維基，並幫助我們瞭解我們的服務的使用。 這些第三方服務提供商可能會使用 cookie 和其他技術來收集有關您使用我們的服務以及其他網站和服務的資訊，包括您的 IP 地址、裝置 ID、檢視的頁面和點選的連結。
